This document describes the syntax supported by Whispem version 0.3.0.

---

## Programs

A program is a sequence of statements written line by line.

---

## Variables

Declared using `let`.

    let x = 10

Variables are immutable.

---

## Print

Outputs the value of an expression.

    print x

---

## Expressions

Supported expressions:

- numeric literals
- variable references

Arithmetic expressions are not supported yet.

---

## Comments

Comments start with `#`.

    # This is a comment
    let x = 10
